## [0.1.0] - 2026-01-18

### Added
- Initial release of vm-tui
- Terminal User Interface (TUI) for VolleyMetrics/Hudl content downloads
- Puppeteer-based automation for content retrieval
- Interactive CLI built with Ink and React
- Support for downloading video and DVW files
- Bun runtime support
